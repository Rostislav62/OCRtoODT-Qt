# Support

## Getting Help
For questions and support:
- Use **GitHub Issues**: https://github.com/Rostislav62/OCRtoODT-Qt/issues

When reporting a problem, please include:
- OCRtoODT version (e.g. `vX.Y.Z`)
- How you installed it (AppImage)
- Your OS / distribution and desktop environment (Wayland/X11)
- Relevant log output (copy/paste from terminal)

## Bug Reports
Before opening an issue:
1. Check existing issues (open and closed).
2. Try running the AppImage from a terminal to capture errors:
   ```bash
   ./OCRtoODT-*.AppImage
   ``` 

## Feature Requests
Feature requests are welcome via GitHub Issues. Please describe:
- the use case
- expected behavior
- why it matters for OCR pipeline workflows

## Commercial Support
At the moment, OCRtoODT is maintained as an open-source project.
If you need prioritized support, open an issue and describe your needs.
