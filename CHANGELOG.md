## [3.0.5] - 2026-03-08
### Added
- First public release.
- Language profiles (create sets of OCR languages and switch profiles quickly).
- Automatic language download when a selected OCR language is missing.

### Fixed
- Fixed OCR behavior when a selected language was missing: progress could jump to 100% as if OCR had already finished.

### Known issues
- Language list and profiles currently show language codes (abbreviations) rather than full human-readable names.
