## Download
- **AppImage:** `OCRtoODT-v3.0.5-x86_64.AppImage`

## What’s new
- First public release of OCRtoODT (deterministic OCR pipeline desktop app).
- Language profiles: create sets of OCR languages and switch profiles quickly.
- Automatic language download: if a selected language is missing, it will be downloaded and then used for recognition.

## Fixes
- Fixed OCR behavior when the selected language was missing: previously progress could jump to 100% as if OCR had already finished.

## Notes / Compatibility
- **Config:** no changes in `config.yaml` (no migration needed).
- **Linux:** AppImage (x86_64).

## Known issues
- Language list and profiles currently show language codes (abbreviations) rather than full human-readable names. This will be improved in a future release.

## Security / trust
- Checksums (SHA256SUMS): planned for upcoming releases.
